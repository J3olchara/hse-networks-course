    targetMAC := net.HardwareAddr(arp.DstHwAddress)

#### Метод расчета в приложении:

**1. Фильтрация фреймов (код: statistics/statistics.go, строки 73-75):**

Учитываются только фреймы между устройством и роутером:
```
(Source MAC == MAC устройства И Destination MAC == MAC роутера)
ИЛИ
(Source MAC == MAC роутера И Destination MAC == MAC устройства)
```

Это реализовано через функции:

- `utils.IsRouterToDevice()` - utils/parser.go, строки 90-92

**2. Расчет размера каждого фрейма (код: utils/parser.go, строки 77-84):**

```go
func CalculateFrameSize(packet gopacket.Packet) int {
    captureLength := packet.Metadata().CaptureLength
    
    // Минимальный размер Ethernet фрейма - 64 байта
    if captureLength < 64 {
        return 64
    }
    
    return captureLength
}
```

**Структура Ethernet фрейма:**

| Компонент | Размер | Учитывается libpcap? |
|-----------|--------|----------------------|
| Preamble | 7 байт | ❌ Нет |
| SFD (Start Frame Delimiter) | 1 байт | ❌ Нет |
| **Destination MAC** | 6 байт | ✅ Да |
| **Source MAC** | 6 байт | ✅ Да |
| **EtherType** | 2 байта | ✅ Да |
| **Payload** | 46-1500 байт | ✅ Да |
| **Padding** | 0-42 байта | ✅ Да (если есть) |
| FCS (Frame Check Sequence) | 4 байта | ⚠️ Зависит от настроек |

**Минимальный размер фрейма:**
- Без Preamble/SFD/FCS: 14 + 46 = 60 байт
- С FCS: 64 байта ← используем как минимум

**В коде:**
- `packet.Metadata().CaptureLength` возвращает размер захваченных данных
- Если это меньше 64 байт, принимаем минимум 64
- Padding включается автоматически (для ARP пакетов Payload = 28 байт, padding = 18 байт)

**3. Суммирование:**

Для каждого направления ведется отдельный счетчик:
- `DataVolumeToRouter` - устройство → роутер
- `DataVolumeFromRouter` - роутер → устройство
- Итого: сумма обоих направлений

**Результаты эксперимента:**

- Устройство → Роутер: [X] байт ([N] фреймов)
- Роутер → Устройство: [Y] байт ([M] фреймов)
- **Всего:** [X+Y] байт

**Верификация:**

Средний размер фрейма: [X+Y]/[N+M] ≈ [Z] байт
- Ожидаемо для ARP: 64 байта (минимальный размер с padding)
- Если больше: возможно учитываются не только ARP пакеты [проверьте код]
